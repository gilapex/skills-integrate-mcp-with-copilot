# Advanced Event Details

## Description
Enhance activity/event data model with more fields for richer information and engagement.

## Requirements
- Add fields: entry fee, budget allocation, poster image, attendance duration, venue details
- Add engagement metrics: view count, interest polls (interested/not interested)
- Support image uploads and display
- Update API and frontend to handle new fields
- Add event categories or tags

## Benefits
- More informative activity listings
- Better user engagement through polls and images
- Support for paid events or budgeted activities

## Related Features
- Builds foundation for budget management and gallery
